# Zettelkasten Workflow

## Processo

1. **Captura**: Ideias, trechos, artigos vão para `slipbox/Inbox/`
2. **Processamento**: Transformar em notas atômicas em `slipbox/`
3. **Conexão**: Usar `[[wikilinks]]` para conectar notas relacionadas
4. **Revisão**: Revisar periodicamente via graph view

## Tipos de Nota

### Nota de Referência (slipbox/References/)
- Resumos de livros, artigos, vídeos
- Formato: `Author-Title.md`
- Metadados no frontmatter quando disponível (kindle highlights, etc)

### Nota de Sermão (slipbox/Sermões/)
- Esboços e conteúdo de sermões
- Links para passagens bíblicas e referências teológicas

### Nota Atômica (slipbox/)
- Uma ideia por nota
- Conectada a outras via wikilinks
- Curta e focada

### Nota de Inbox (slipbox/Inbox/)
- Captura rápida, não processada
- Mover para tipo correto após processamento

## Tags Usadas

## Convenções
- Título descritivo e único
- Wikilinks para notas relacionadas
- Manter notas curtas (< 500 palavras idealmente)
