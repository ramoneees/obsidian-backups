# Memory Chips and China, Microsoft and Chinese Models

> **Nota:** artigo atrás do paywall (Stratechery Plus). Apenas o lead está publicamente disponível.

## Lead (público)

> The big three memory makers may come to regret opening up the door to Chinese memory makers; Microsoft, meanwhile, is very incentivized to use Chinese models.

## Teses Implícitas (a partir do lead)

1. **Memory chips** — Samsung, SK Hynix, Micron podem se arrepender de ter aberto a porta para fabricantes chineses de memória. Implicação: precedente para semicondutores em geral, não apenas chips lógicos.
2. **Microsoft + modelos chineses** — Microsoft tem forte incentivo para usar modelos chineses. Possíveis razões: custo, acesso ao mercado chinês, diversificação de fornecedores, dinâmica competitiva vs OpenAI/Anthropic.
