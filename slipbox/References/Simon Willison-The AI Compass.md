# The AI Compass

## Summary

Link post: Simon Willison highlights a political-compass-style quiz (The AI Compass) by bambamramfan. 29 questions about AI and AI ethics place the user in one of 30 archetypes. Simon's first-pass result: "The Garage Tinkerer" (patron saint: Simon Willison himself). Single-page React app using the `<script type="text/babel">` trick to avoid the build step.

## Key Ideas

- **The quiz** by [bambamramfan](https://bambamramfan.tumblr.com/post/820505178072580096/the-ai-compass) — political compass style with axes GOOD / BAD and OVERHYPED / TRANSFORMATIVE. 30 archetypes.
- **Simon's archetype:** "The Garage Tinkerer" — *"You're running local models, building little tools, and having a genuinely great time. You don't care about the discourse — you care about making the thing do cool stuff."*
- **Implementation note:** single-page React app using `<script type="text/babel">` to avoid the build step. [Code here](https://github.com/bambamramfan/ai-compass/blob/main/index.html).
- **Implied commentary:** a fun taxonomy of how people approach AI. Useful for self-reflection — which archetype am I, which one is my team, which one is my audience?

## Aplicação Prática

- Faça o quiz você mesmo para autoconhecimento sobre postura em AI.
- Para times: pode ser um bom exercício de discussão — diferentes arquétipos no time geram tensões produtivas sobre estratégia de AI.
- Para autores/conteudistas: a taxonomia pode ser referência para escolher tom e abordagem.
