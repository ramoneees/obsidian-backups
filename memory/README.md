# 🧠 Hermes Extended Memory

**Sync**: CouchDB LiveSync (primary) — https://obsidian.ramoneees.com
**Backup**: GitHub (fallback) — https://github.com/ramoneees/obsidian-backups.git

> Notes created here sync automatically via LiveSync. No manual git commits needed.

---

## Structure

```
memory/
├── systems/          # How things work (APIs, configs, workflows)
├── people/           # Contacts, relationships, context
├── projects/         # Ongoing work, history, decisions
├── knowledge/        # Learnings, patterns, references
└── sessions/         # Session logs for cross-session recall
```

## Principles

- **Don't duplicate** what's already in Hermes memory
- **Go deeper** — detailed docs, examples, edge cases
- **Link liberally** — wikilinks between related notes
- **Update on change** — this is live, not archival
